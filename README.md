# sales-management

### 1. 기여도
- 화면 디자인, DB설계, 개발 100%

### 2. 개발 기간
- 2022.10 ~ 2023.03 (총 6개월)

### 3. 주요 기능
- 로그인 : 활성/비활성/관리자 계정 구분, 로그아웃
- 공지사항 : 보기, (관리자)등록/수정/삭제
- 거래처 : 등록/수정/삭제
- 시트(TGP) : 등록/수정/미리보기
- 계정관리 : (관리자)등록/수정/삭제
